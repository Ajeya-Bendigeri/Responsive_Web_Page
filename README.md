# Responsive_Web_Page
This project is a responsive web page implementation created using HTML, CSS, and JavaScrip based on the design provided in the Figma file.

The goal of this assignment is to accurately replicate the design specifications while implementing interactive functionality such as a sticky header and image carousel with zoom feature, ensuring responsiveness across different screen sizes.

Figma Design
Design reference used for implementation:

Figma Link:
https://www.figma.com/design/DOv07H7C2tA5UrVLhmfwfW/Gushwork-Assignment?node-id=490-8785&t=Z0PPuWCdxPbNLcSw-1

The layout and components were implemented to match the design as closely as possible.

Features Implemented
1. Responsive Layout
2. Sticky Header
3. Image Carousel
4. Image Zoom on Hover
5. Clean Code Structure
Tech Stack
HTML5
CSS3
JavaScript
